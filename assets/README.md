# Assets

Здесь должны быть следующие файлы:

- `icon.png` - иконка приложения (1024x1024)
- `splash.png` - splash screen (1242x2436)
- `adaptive-icon.png` - адаптивная иконка для Android (1024x1024)
- `favicon.png` - favicon для веб (48x48)
- `notification-icon.png` - иконка для уведомлений (96x96)

Для MVP можно использовать временные изображения или создать их через онлайн-генераторы.
